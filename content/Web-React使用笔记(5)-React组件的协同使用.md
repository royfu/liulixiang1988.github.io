##1. 组件协同使用介绍

组件的协同就是对组件的一种**组件、管理的方式**。

目的：

- 逻辑清晰
- 代码模块化
- 封装细节
- 代码可复用

组织方式：

- 组件嵌套：实现封装
- Mixin:直接嵌入代码，实现复用

##2. 组件嵌套

组件嵌套的本质是**父子关系**。

父组件和子组件是如何通信的呢？
父组件给子组件通信是用属性。
子组件对父组件通信是：父组件把事件处理函数作为属性传递给子组件，子组件再进行调用。
