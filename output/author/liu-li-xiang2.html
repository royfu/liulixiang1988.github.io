Flask笔记－01 基本应用框架

六 18 十月 2014

1. 初始化

from flask import Flask

app = Flask(__name__)

__name__的作用是用来给Flask来判断应用的根路径的。

2. 路由和视图函数

@app.route('/')
def hello_world():
    return 'Hello World!'
